# coffemania_testproject
Тестовое задание для кофемании от 30.03.2026

Проект предназначен для расчёта расстояния между двумя точками. Входные параметры: две точки на плоскости, заданные в double формате. Выходные данные: расстояние в double формате, либо сообщение об ошибке.

Проект может быть использован двумя способами:

1. встраиваемая библиотека dll (nuget-пакет) для платформы .Net
2. сервис с Web API интерфейсом

Архитектурные решения: $/documentation/Архитектура.docx
